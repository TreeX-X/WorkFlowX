/xwhole
全局模式
新功能、跨模块重构。完整规划-评估循环，生成 Hybrid Tree 需求文档。
需求发现→ 规划对话→ 编码实现→ 评估审计→ 迭代
  • 苏格拉底式需求发现 + 主动质疑
  • 多轮规划对话
  • Hybrid Tree（Parent + Children）
  • AC 交叉验证，最多 N 轮
  • Worktree 隔离
全平台支持
/xwhole -parallel
并行模式
多个独立子任务并行执行。Agent Teams 实现并行编码与评估。
规划→ 创建团队→ 并行执行→ 合并
  • 与 xwhole 相同的规划流程
  • 自动拆分为并行任务
  • 依赖图自动调度
  • 多个 coder/evaluator 队友
  • 双层隔离保护
仅 Claude Code
/xlocal
局部模式
1-2 个模块内的修改。跳过完整 PRD 规划，使用轻量级需求发现。
发现→ PRD→ 编码→ 评估→ 迭代
  • 轻量级需求发现（Module 08）
  • 自动检测或生成 Hybrid Tree
  • promptMasterX 优化
  • evaluatorX，最多 N 轮
  • Worktree 隔离
全平台支持
/xunit

优化→ 编码→ 完成
  • promptMasterX 优化
  • coderX 轻量执行
  • 仅 Karpathy 指南
  • 无 Bus Payload
  • 最快交付
